TMU Dashboard v28 - UI, export, notes, and comparison cleanup

Changes in v28:
1. Chart title now defaults to well name(s) only, without date.
   - Single well: Well B15-42
   - Multiple wells: B15-42 vs B3C18-7
   - User can override the title from the sidebar.

2. Removed UI items requested by user:
   - Parser build caption
   - Full range time mode
   - Overlay normalized plot style
   - Forward fill and Time interpolation fill methods
   - Download cleaned Excel
   - Download filtered CSV
   - Plotly/Kaleido fallback captions
   - Human-readable report exports heading/caption
   - ZIP download for PNG charts

3. Download buttons renamed to:
   - Download single chart PNG
   - Download single chart PDF
   - Download separate charts PNGs
   - Download multi-charts PDF

4. PNG/PDF downloads are generated by Matplotlib for consistent, high-resolution output.
   This avoids showing Kaleido/Chrome fallback warnings to the user.

5. Notes now appear in every chart/panel:
   - Point notes show as vertical labels in each chart.
   - Interval notes show start/end dashed lines plus the interval label inside each chart.

6. Note deletion is clearer:
   - Select one or multiple point notes and remove them.
   - Remove all point notes.
   - Select one or multiple interval notes and remove them.
   - Remove all interval notes.

7. Multi-well compressed date comparison readability improved:
   - Fewer x-axis tick labels.
   - Fewer value labels in comparison mode to reduce overlap.
   - Well labels remain in the legend.

Run:
pip install -r requirements.txt
streamlit run app.py
